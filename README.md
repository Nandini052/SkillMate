# SkillMate 🔄

A peer-to-peer skill exchange platform where users can connect, teach, and learn from each other.

## Features
- 👤 User authentication & profiles
- 🔍 Smart skill matching
- 💬 Real-time chat
- 🤝 Connection requests
- 🏠 Community discovery

## Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS for styling
- Firebase for data storage and real time updates
- Feather Icons

## Getting Started
1. Open `index.html` in your browser
2. Use test credentials or sign up
3. Start connecting with other users!
